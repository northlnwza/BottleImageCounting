# Bottle Image Counting

This project performs automated bottle counting from images using computer vision techniques.

## Features
-   Image processing with OpenCV.
-   Automated detection and counting.
-   Interactive Jupyter Notebook for easy use.

## Getting Started

### Prerequisites
Ensure you have Python installed. Install the required dependencies:

```bash
pip install -r requirements.txt
```

### Usage
Run the Jupyter Notebook to analyze your images:
-   `DetectFromImage_BottleImageCountingipynb.ipynb`
